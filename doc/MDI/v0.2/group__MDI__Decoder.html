typedef MDI_object_t MDI_Decoder_t

Decoder object abstraction.

All decoding operation, i.e. construction of Operations from encoded execution buffers are handled through this object.

Decode a buffer into an Operation.

Given the buffer and buffer_size to an encoded stream, decode current_ptr into an Operation and update current_ptr.

The current_ptr reference must point to a position inside the given buffer. It will not be updated if the decode point is an invalid Opcode. Otherwise if a valid Opcode if found, the decoder will construct the associated Operation and update the current_ptr reference to the next decoding point.

Parameters
selfA Decoder.
bufferAn encoded buffer pointer.
buffer_sizeThe size of the encoded buffer.
current_ptrThe reference to the current decoding pointer.
Returns
A new Operation instance or NULL on invalid Opcode..
