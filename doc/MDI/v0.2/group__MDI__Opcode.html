Detailed Description

Proxy to a MDD Opcode description.

Typedef Documentation

typedef MDI_object_t MDI_Opcode_t

Opcode class abstraction.

Should give more information on encoding for an Instruction when multiple processors of the same family are implemented. Otherwise there is a 1-1 match with Instruction.
