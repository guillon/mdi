Detailed Description

An abstract object for Operation, an executable Operator instance.

Typedef Documentation

typedef MDI_object_t MDI_Operation_t

Operation object abstraction.

An Operation is an executable instance of a MDI Operator. Decoders create new Operations from binary buffers. Assemblers create new Operations from assembly buffers. Disassemblers output assembly forms of Operations, Executions execute Operations, Encoders output encoded Operations into binary buffers.
