Detailed Description

Machine Description Interface Operations, C abstract interface.

The Operations interface provides access to executable architecture Operators as defined in the MDI interface.

An executable Operator is abstracvted into an Operation instance from which a Encoders, Disassemblers, Executions can work on. Operations are created from Decoders and Assemblers.
